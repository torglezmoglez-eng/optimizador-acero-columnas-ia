# optimizador-acero-columnas-ia
El presente proyecto propone el desarrollo de un asistente inteligente para el cálculo de la cuantía de acero en columnas. El sistema solicitará al usuario las dimensiones de la columna (largo y ancho en centímetros), calculará el área bruta del concreto y determinará el área mínima de acero requerida considerando que debe ser equivalente al 1%
